## 13:11 - Build EPERM fix

**Problem:** `npm run build` failing with `EPERM: Permission denied` on `dist\assets` — Avast antivirus holding file handles after previous build.

**Fix:** Added `emptyOutDir: false` to `build` config in `vite.config.js` (same Avast workaround pattern as `.vite-cache`).

**Side effect:** Stale hashed chunks accumulate in `dist/assets` over time — harmless but can be manually cleared once Avast exclusions are set.
