## World Clock & Time Zone Converter (Phase 25, task 38)
- **Tool:** `src/tools/reference/world-clock.js` - live multi-zone wall clocks + cross-zone date/time converter, all IANA aware via `Intl.DateTimeFormat`.
- **Features:**
  - Live ticking wall clocks for added zones (re-renders time + date every second)
  - Add via typed IANA string (with `<datalist>` autocomplete) or quick-pick chip
  - Reorder (↑/↓), remove, set-as-base, reset to defaults
  - localStorage persistence of zones + baseZone
  - Time converter: pick base zone + date/time, see equivalent local time in every listed zone
  - "Use now" button to seed date/time inputs from current clock
  - DST-aware offsets (computed for the exact moment in view, not a static number)
- **Exported helpers:** `getAllTimeZones`, `getZoneOffsetMinutes`, `formatInZone`, `formatDateInZone`, `convertTime`, `buildSourceUTC`, `getDefaultZones`
- **Design decision — `convertTime` semantics:** sourceDate's UTC components are interpreted as wall clock in `sourceZone`; function returns the true UTC instant by subtracting `sourceOffset`. Pairs with `buildSourceUTC` in render (date+time inputs → Date via `Date.UTC`).
